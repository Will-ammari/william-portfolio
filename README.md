# William Ammari Portfolio

A personal portfolio website for William Ammari, focused on Laravel backend development, API-first SaaS products, AI integration, automation workflows, and compliance-aware business systems for German and European market positioning.

## Positioning

Primary target roles:

- Backend Developer
- Laravel Developer
- API Engineer
- AI Integration Engineer
- Automation Workflow Developer

Main proof points:

- ProcurePilot AI - procurement SaaS backend for German/European SMEs
- KlinikLedger - healthcare practice operations REST API
- FlowGuard AI - AI workflow risk mapping and reporting platform

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Vercel-ready configuration

## Quick Start

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Push this project to GitHub.
2. Go to Vercel.
3. Import the GitHub repository.
4. Framework preset: Vite.
5. Build command: `npm run build`.
6. Output directory: `dist`.
7. Deploy.

## Update Content

Most portfolio content is in:

```text
src/main.jsx
```

Styles are in:

```text
src/styles.css
```
